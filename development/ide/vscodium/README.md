# VSCodium

All the content should work in vscode too.

## Useful keyboard shortcuts

| Keybind | Description |
| --- | --- |
| `CTRL+P` | Opens dialog to search for a file by name |
| `CTRL+0` | Shows the current open file in the file explorer |
