# Git Bash

## Install extension like jq

```bash
mkdir -p "${EXEPATH}/usr/local/bin"
curl -L -o /usr/local/bin/jq.exe \ 
    https://github.com/stedolan/jq/releases/latest/download/jq-win64.exe
```
