# Using a CD

## Ripping audio cds

### By using whipper

ref: https://github.com/whipper-team/whipper

A configuration file is available in: `~/.config/whipper/whipper.conf`

```bash
# Analyze drive
whipper drive analyze

# Rip
whipper cd rip
```
