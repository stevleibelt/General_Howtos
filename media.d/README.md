# Media

## Metadata

```bash
# List metadata
mat2 -s <string: path_to_the_file>

# Remove metadata
#  Creates a cleaned file, keeps the original file as it is
mat2 <string: path_to_the_file>
#  Overwrites original file
mat2 --inplace <string: path_to_the_file>
```
