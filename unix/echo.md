# options

* -n    -   new new line at the end
* -e    -   enable backslash escape
    * \\    -   backspace
    * \a    -   alert (acoustic/bell)
    * \b    -   backspace
    * \c    -   no further output (equals n)
    * \e    -   escape
    * \f    -   form feed
    * \n    -   new line (jump to beginning of new line)
    * \r    -   carriage return (jump to beginning of current line)
    * \t    -   horizontal tab
    * \v    -   vertical tab
