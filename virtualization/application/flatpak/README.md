# application

```
#install
flatpak install <path or url> <name>

#update
flatpak update <name>

#remove
flatpak uninstall <name>
```


# repository

```
#add
flatpak remote-add <name> <path or url>

#remove
flatpak remote-delete  <name>

#list available applications
flatpak remote-ls <path or url>
```
